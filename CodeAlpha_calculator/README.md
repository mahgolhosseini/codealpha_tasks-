# Calculator 🧮

## 📝 Project Description

A responsive web-based calculator that performs basic arithmetic operations including addition, subtraction, multiplication, and division. The calculator features an interactive button interface, a display screen for input and results, and keyboard support for faster calculations. Built using HTML, CSS, and JavaScript, this project demonstrates DOM manipulation, event handling, and real-time calculation functionality.

## 🚀 Features

Perform basic arithmetic operations (+, −, ×, ÷)

Interactive button-based user interface

Real-time display of user input and results

Clear (C) button to reset the screen

Delete (⌫) button to remove the last character

Keyboard input support for faster calculations

Responsive and modern UI design

## 💻 Technologies Used

HTML5

CSS3 (Grid layout, Flexbox, Styling effects)

JavaScript (ES6)

DOM Manipulation

Event Listeners for button and keyboard input
